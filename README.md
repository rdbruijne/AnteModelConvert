# AnteModelConvert
Model conversion tool for [Ante](https://gitlab.com/TropicalTrevor/ante)

## Usage

Drop a model file onto the executable

`AnteModelConvert.exe -o[path] [filepath_0] [filepath_1] ... [filepath_n]`

| Argument | Usage | Description |
|----|----|----|
| `-?` | `-?` | Print command line info |
| `-o` | `-o[directory]` | Override output directory (optional) |

## File format

File format can be found at https://gitlab.com/TropicalTrevor/ante/-/blob/master/FILE_FORMATS.md
